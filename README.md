# sloshdb
Real-time in-memory key-value database inspired by redis written in Typescript

## How to install and run, stop etc

### Requirements
- Git
- Node v14 or above
- PM2, install by running `npm install --global pm2`

### Install Dependencies
```sh
$ npm install
```
